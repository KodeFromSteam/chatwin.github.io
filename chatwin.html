Hypertext Markup Language (HTML) is the standard markup langauge for documents designed to be displayed in a web broswer.

It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as Javascript.

Inventor of HTML

In 1989, Tim Berners-Lee wrote a memo proposing an Internet-based hypertext system. Berners-Lee specified HTML and wrote the broswer and server software in the late 1990.
